这里的信息主要来源于MIPS R1000 Microprocessor User's Manual。

Processor Modes

Kernel mode

Can access and change any register. The innermost core of the operating system runs in kernel mode.

Supervisor mode

Has fewer privileges and is used for less critical sections of the operating system.

User mode

Prevents users from interfering with one another.
